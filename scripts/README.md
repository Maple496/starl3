# Scripts 目录说明

本目录包含 StarL3 主程序的核心代码。

## 目录结构

```
scripts/
├── core/              # 核心引擎模块
│   ├── pipeline_engine.py   # 流程执行引擎
│   ├── registry.py          # 操作注册中心
│   ├── safe_eval.py         # 安全表达式求值
│   └── ...
├── ops/               # 操作模块
│   ├── elt_ops.py           # ETL数据处理
│   ├── file_ops.py          # 文件操作
│   ├── datavisual_ops.py    # 数据可视化
│   └── ...
├── main_starl3.py     # 主程序入口（CLI）
└── main_edit.py       # Web编辑器入口
```

## 重要提示

**本目录只保留与主程序直接相关的代码：**

✅ **可以放入的文件：**
- 主程序入口文件
- 核心引擎模块
- 操作模块（ops）
- 配置文件

❌ **不要放入的文件：**
- 测试相关代码（请放到 `tests/` 目录）
- 数据生成工具（如创建emoji数据等）
- 临时调试脚本
- 压力测试脚本
- 修复/维护脚本（用完即删）

## 维护规范

1. 定期清理临时脚本
2. 测试代码统一放到 `tests/` 目录
3. 工具脚本用完及时删除或移出本目录
4. 保持核心目录整洁
